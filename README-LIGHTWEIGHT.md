# OsteoPerio Lightweight No-Media Build

This package is a lighter website version prepared without the original heavy image and video files.

## What changed
- Rewrote the main website pages with a more patient-first premium structure
- Replaced the old Projects concept with an International Patients page
- Simplified the site so it does not depend on the large media assets
- Added a basic consultation POST route at `/api`
- Updated navigation, footer, WhatsApp button, metadata, robots, and sitemap files
- Added hash-based internal page navigation so sections/pages are easier to open directly

## Before launch
- Replace placeholder clinic phone, WhatsApp, address, and legal details
- Connect the consultation route to your real email inbox or CRM
- Add final biographies, credentials, and any optimized media you want later
- Test the site locally after installing dependencies in your normal environment
